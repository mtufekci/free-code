<objective>
Translate Ollama tool call responses into Anthropic streaming event format for proper tool execution handling.
</objective>

<purpose>
When Ollama returns a tool call (detected via `message.tool_calls` or `done_reason: "tool"`), the response must be translated to Anthropic streaming format: `content_block_start` with `tool_use` type, `content_block_delta` with `input_json_delta`, and `content_block_stop`. This allows the existing streaming infrastructure to handle tool calls correctly.
</purpose>

  <name>task 1: Add tool call response translation to translateEventToAnthropic()</name>
  <files>src/services/api/ollama.ts</files>
  <action>
Update `translateEventToAnthropic()` (around line 204) to detect and translate tool calls:

1. Check if `event.message.tool_calls` exists and has entries
2. If tool call detected:
   - First `content_block_start`: emit with type `tool_use`, name from `tool_calls[0].function.name`, and empty `input`
   - `content_block_delta`: emit with type `input_json_delta` and `partial_json` containing the parsed arguments
     - Note: Ollama returns `arguments` as a JSON string (e.g., `'{"city":"Tokyo"}'`) — parse it and emit the partial_json
     - Handle both string and object format for arguments (some Ollama versions return object)
   - `content_block_stop`: emit to close the tool block
   - Skip the text content block emission when tool call is present
3. If no tool call, emit current text-based content block (existing behavior)

For arguments parsing:
- Ollama sends arguments as JSON string in `tool_calls[0].function.arguments`
- Parse with JSON.parse() wrapped in try/catch
- If parse fails, emit partial input with empty string and log warning
- If arguments is already an object (some versions), use it directly

Ollama tool call response example:
```json
{
  "message": {
    "role": "assistant",
    "content": "",
    "tool_calls": [{
      "function": {
        "name": "get_weather",
        "arguments": "{\"city\":\"Tokyo\"}"
      }
    }]
  },
  "done_reason": "tool",
  "done": true
}
```

Anthropic streaming format needed:
```typescript
{ "type": "content_block_start", "index": 0, "content_block": { "type": "tool_use", "name": "get_weather", "input": "" }}
{ "type": "content_block_delta", "index": 0, "delta": { "type": "input_json_delta", "partial_json": "{\"city\":\"Tokyo\"}" }}
{ "type": "content_block_stop", "index": 0 }
```
