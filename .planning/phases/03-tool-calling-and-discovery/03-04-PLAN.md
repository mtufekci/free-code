<objective>
Integrate model capability detection to skip tools for models that don't support them.
</objective>

<purpose>
Some Ollama models do not support tool calling. Before including tools in a request, the system must check if the current model supports tools via `supportsTools` from `extractContextWindow()`. If the model doesn't support tools, tools should be silently excluded from the request (not an error — graceful degradation).
</purpose>

</context>
