<objective>
Enforce context window limits in Ollama requests to prevent silent truncation.
</objective>

<purpose>
Ollama's `num_ctx` option sets the context window size. Without it, Ollama may silently truncate conversations that exceed its default context. Additionally, the system must detect when context window is exceeded (`done_reason: "model_full"`) and handle it gracefully with user-visible feedback.
</purpose>
