## Context

The Ollama client must **act as an interceptor** - mimicking the Anthropic SDK interface exactly so existing code in `claude.ts` works without modification. The key insight from research:

> The Ollama client must mimic the Anthropic SDK interface exactly (`client.beta.messages.create()` returning `Stream<BetaRawMessageStreamEvent>`) so existing code works without modification. Translation happens in real-time as Ollama SSE events stream in.

## Architecture

```
Anthropic SDK call style:
  client.beta.messages.create(params, { signal }).withResponse()
       ↓
Our Ollama client must provide the SAME interface:
  ollamaClient.beta.messages.create(params, { signal }).withResponse()
       ↓
Translates params (Anthropic → Ollama format)
       ↓
Calls Ollama /api/chat with SSE streaming
       ↓
Translates SSE events (Ollama → Anthropic format) in real-time
```

### Key Implementation Details

**Request Translation (`translateRequestToOllama`):**

```typescript
function translateRequestToOllama(params: Parameters<Anthropic['beta']['messages']['create']>[0]): OllamaChatRequest {
  return {
    model: params.model,
    messages: params.messages.map(msg => ({
      role: msg.role === 'user' ? 'user' : msg.role === 'assistant' ? 'assistant' : 'user',
      content: 'content' in msg ? msg.content : '',
    })),
    stream: true,
    options: {
      temperature: params.temperature,
      num_predict: params.max_tokens,
    },
  }
}
```

**Event Translation (`translateEventToAnthropic`):**

| Ollama Event | Anthropic Event |
|--------------|-----------------|
| First chunk | `message_start` |
| `message.content` delta | `content_block_delta` (text_delta) |
| `done: true` | `message_delta` + `message_stop` |

**SSE Parsing:**

```typescript
function* parseSSEStream(response: Response): Generator<OllamaChatResponse> {
  const reader = response.body!.getReader()
  const decoder = new TextDecoder()
  let buffer = ''

  for await (const chunk of reader) {
    buffer += decoder.decode(chunk, { stream: true })
    const lines = buffer.split('\n')
    buffer = lines.pop() ?? ''

    for (const line of lines) {
      if (line.startsWith('data: ')) {
        yield JSON.parse(line.slice(6))
      }
    }
  }
}
```

**The Client Interface:**

```typescript
export function createOllamaClient(): {
  beta: {
    messages: {
      create: (params: MessageCreateParams, options?: { signal?: AbortSignal }) => {
        return {
          withResponse: () => Promise<{ data: AsyncGenerator<BetaRawMessageStreamEvent> }>
        }
      }
    }
  }
}
```
